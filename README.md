# opencode-config

## Install OpenCode

Follow the official OpenCode installation instructions:

```text
https://opencode.ai
```

Check that OpenCode is installed:

```bash
opencode --version
```

## Install This Config

Clone this repository:

```bash
git clone https://github.com/w0rp/opencode-config.git ~/.config/opencode
```

Authenticate as normal, and that's it!
